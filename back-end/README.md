# Backend - Pro Bono Legal Aid App

## Overview
This is the **backend** for the Pro Bono Legal Aid App, built with **Node.js, Express.js, and MongoDB**. It handles user authentication, case management, and lawyer matching.

## Features
- User and lawyer authentication (JWT)
- API endpoints for legal aid requests
- Lawyer profile management
- MongoDB integration for storing cases and users

## Tech Stack
- **Node.js**
- **Express.js**
- **MongoDB**
- **Mongoose (ODM)**
- **JWT Authentication**
